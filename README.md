#SnackMatch

SnackMatch is a web application that helps users find and order their favorite snacks based on their mood and preferences. Whether you're bored, hungry, or angry, SnackMatch has got you covered!

## Features
- Mood-based snack recommendations
- User reviews and ratings
- Easy online ordering
- Delivery tracking
- Personalized snack profiles

## Installation
1. Clone the repository:
    ```
    git clone  https://github.com/ShenzaMariyam/SnackMatch.git
    ```
2. Navigate to the project directory:
    ```
    cd SnackMatch
    ```
3. Install the required dependencies:
    ```
    npm install
    ```
4. Start the development server:~
    ```
    npm start
    ```
5. Open your browser and go to http://localhost:3000 to view the application.
